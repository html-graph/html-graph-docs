## Unmark Port

Unmarks specified port. All the edges adjacent to port get removed.

{{< code lang="javascript">}}
canvas.unmarkPort("port-1")
{{< /code >}}

### `unmarkPort` Parameters

| Name          | Type                                                          | Description                  | Required |
|---------------|---------------------------------------------------------------|------------------------------|----------|
| id            | any                                                           | Identifier of port to unmark | yes      |
